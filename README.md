## Развлекательный портал на Flask

Мини‑портал: лента постов, категории, поиск, страница поста, комментарии, регистрация/логин, профиль, создание постов, простая админ‑модерация.

### Быстрый старт (Windows / PowerShell)

Создайте виртуальное окружение и установите зависимости:

```bash
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

Запуск:

```bash
python app.py
```

Откройте в браузере: `http://127.0.0.1:2222`

### Переменные окружения (опционально)

Можно создать файл `.env` рядом с `app.py`:

- `SECRET_KEY` — секретный ключ Flask
- `DATABASE_URL` — например `sqlite:///portal.db`
- `ADMIN_EMAIL` — email, который получит роль админа при первом запуске (если пользователь с таким email зарегистрируется)
